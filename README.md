# Название проекта
Анализ успеваемости студентов

Фамилия Группа  
Мауль Д.Е., МПБ-301-0-01

Краткое описание:  
Проект предназначен для анализа факторов, влияющих на успеваемость студентов, и прогнозирования среднего балла (GPA) с использованием модели множественной линейной регрессии.

## Стек
- Язык и фреймворк: Python
- Ключевые библиотеки: pandas, numpy, scikit-learn
- Сервисы и интеграции: Git, GitHub
